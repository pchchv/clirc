# clirc IRC Terminal Client

clirc is a sleek, minimal IRC client for terminal.  
Chat faster, cleaner, and without distractions right from terminal.

### Installation

```
go install github.com/pchchv/clirc
```

### Usage

Open a fresh terminal and type:
```sh
clirc
```

Enter IRC details and pick favorite channels.

### Keybindings

| Key     | Action            |
|---------|-------------------|
| ↑/↓     | Scroll chat       |
| ←/→     | Switch panes      |
| Ctrl+C  | Quit              |
